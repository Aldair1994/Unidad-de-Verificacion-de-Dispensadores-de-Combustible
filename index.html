Croquis de la estación

Agregar caja (productos por lado):
Área del croquis · 24×24
▲ OFICINA ADMINISTRATIVA
100%
Las cajas se rotulan A, B, C… y las mangueras se numeran solas según la posición (de adentro/arriba hacia afuera y de izquierda a derecha). Arrastra el lienzo con un dedo para desplazarte y pellizca (o usa − / +) para acercar/alejar, así puedes agregar cajas hacia los lados. Para mover una caja usa su asa ⠿; ↻ la gira y ✕ la elimina. En cada manguera elige el producto: D (Diesel), R (Regular), S (Superior), VP (V Power), ULS (ULS Diesel), ION (ION Diesel), JET (Jet A), KER (Kerosene). El PDF usa una cuadrícula fija de 24×24, así que mantén las cajas dentro de ese espacio para que no se recorten.

Ajustes y sincronización

Sincronización en tiempo real

Pegue la URL del servidor que recibe los datos (endpoint POST JSON). Al guardar, cada verificación se envía automáticamente; si no hay señal, queda en cola y se reintenta al recuperar conexión.

¿Cómo crear el servidor gratis con Google Sheets?
  1. Cree una hoja en Google Sheets.
  2. Menú Extensiones → Apps Script.
  3. Pegue una función doPost(e) que haga appendRow con JSON.parse(e.postData.contents).
  4. Implementar → Nueva implementación → Aplicación web, acceso “Cualquiera”.
  5. Copie la URL /exec y péguela arriba.

Así los datos llegan a la hoja al instante y se actualizan en tiempo real para toda la unidad.
